# Beldex Address Generator (Python)

A simple Python script to generate **Beldex wallet addresses** programmatically.  
This project demonstrates how developers can create Beldex addresses for testing, development, or educational purposes.

---

## Features

- Generate Beldex wallet addresses
- Lightweight and easy to use
- Simple Python implementation
- Suitable for development and testing
